### 2️⃣ Calculadora de Partidas Rankeadas
* **Objetivo:** Calcular o saldo de partidas rankeadas e classificar o rank do herói.
* **Diferencial:** Implementação da lógica utilizando o conceito real de **MMR** (Matchmaking Rating) dos games, onde a pontuação de rank é calculada com base no saldo de vitórias e derrotas.
* **Conceitos aplicados:** Funções, Retornos, Variáveis e Operadores Aritméticos.
